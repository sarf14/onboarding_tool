## **How to Use This Template**

1. **Access the Google Sheet:** https://docs.google.com/spreadsheets/d/11XiHURNsFLm4NjW-l5o8SMdepQ4ZPWCoynAuBDA-W0s/edit?gid=0#gid=0

2. **For each day:**
   - Open the corresponding tab (Day 1 Ans, Day 2 Ans, etc.)
   - Copy the task URLs from that tab
   - Paste them into the appropriate day section above

3. **Task Selection Guidelines:**
   - **Day 1-2:** Select simpler, introductory tasks
   - **Day 3-4:** Select moderate complexity tasks
   - **Day 5:** Select real tasks of varying complexity
   - **Day 6:** Select H2H tasks focused on outcome evaluation
   - **Day 7:** Select complete H2H evaluation tasks

4. **Distribution Rules:**
   - Start with fewer tasks (2-3) on early days
   - Gradually increase task count (3-5) on later days
   - Ensure variety in task types
   - Include both simple and complex examples
